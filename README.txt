The Dax Toolkit provides a collection of data analysis and visualization
algorithms that run well on multi- and many-core processors.  The Dax
Toolkit also makes it easy to design other visualization algorithms that
work well on such processors.

For more information about Dax including how to install and use the
toolkit, please visit our website at http://daxtoolkit.org.
